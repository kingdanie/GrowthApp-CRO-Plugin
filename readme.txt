A WordPress plugin that helps track and improve website conversion. 
  
== Description ==
  
Growth CRO AI is your AI-powered Conversion Rate Optimization (CRO) assistant that automates tasks and helps you improve conversions on your WordPress website.

== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
  
== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
1. Install the plugin via WordPress reposiotry.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the GrowthApp CRO AI settings page.
4. Vist your account at https://my.growthapp.io and copy the paste your GrowthApp Project ID and API Key obtained from the GrowthApp web portal.
Save changes.
  
= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
